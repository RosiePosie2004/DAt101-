1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. I would use constants for values that never change, like pi or a fixed number, and variables for values that can change, like a game score. Not everything can be constant because some things need to change.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses change the order of operations in math. The numbers inside the parentheses is calculated first, which can change the result.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We can limit floating-point numbers to a certain number of decimals by rounding them, like using methods that let us show only two decimals.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. Because floating-point numbers can be slightly off, we cant always check if they are exactly equal. We check if they are close enough instead, for example by seeing if the difference is very small.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators give back either true or false. We can check if they are true or false by printing them or using them in an if statement.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() gives a random number between 0 and 1. It can be used for games, picking something randomly, or simulating chance.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. Modulus (%) gives the remainder when you divide one number by another. We use it to check things like whether a number is even or odd, or to divide things into groups.
